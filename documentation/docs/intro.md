# Introduction
 
 
`akash-build` is a command-line tool that simplifies the process of building and pushing container images to Docker Hub using [Pack](https://buildpacks.io/) and [Docker](https://www.docker.com/). With this tool, you can easily containerize your applications and make them available on Docker Hub for others to use.
